# tidybot_platform
A codebase for the ROS interface and other tools for controlling the tidybot++ mobile manipulator

## tidybot_description
The package that contains the urdf/xacro and CAD resources for the tidybot++. It also defines the way how to spawn the tidybot++ model in Gazebo

## tidybot_moveit_config
The package is generated by the moveit setup assistant and defines how to interact with MoveIt2

## tidybot_control
The package which provides nodes for joystick teleop / webapp teleop

## tidybot_msgs
The package which defines the tidybot++ custom messages
